RecidivismData_Original.csv includes all the data extracted from the propublica research data which contains 6,890 users' recidivism information.

In this data,there are three columns: Violence_score, Recidivism_score and MarriageStatus which are extracted from original data resources based on the name and dob match.


RecidivismData_Normalized.csv includes all the same data whose attributes have been normalized by formula: (val- minimum)/(maximum - minimum). Also all the categorical attributes in the RecidivismData_Original.csv have been mapped into numerical value. The mapping information is listed in the corresponding mapping files under the same folder.
