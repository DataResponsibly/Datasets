The “LendingClubCreditData” dataset hosted on Lending Club contains approximately 115 attributes. 
This file contain complete loan data for all loans issued through the time period stated, including the current loan status (Current, Late, Fully Paid, etc.) and latest payment information. The file containing loan data through the "present" contains complete loan data for all loans issued through the previous completed calendar quarter. It includes approximately 42,538 observations, with 115 attributes.
Another is rejected record data which include 755,423 observations with 8 attributes. This file contains the list and details of all loan applications that did not meet Lending Club's credit underwriting policy.

The most attributes in Loan data are numerical. Since there are so many attributes, please see the “README.xlsx” file which include all the attributes’ description.
